# Knowledge Base Builder

Build searchable knowledge bases

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/knowledge-base-builder
cd knowledge-base-builder
pip install -e .
```

## Usage

### Python

```python
from knowledge_base_builder import KnowledgeBaseBuilder

result = KnowledgeBaseBuilder.process("data")
print(result)
```

### CLI

```bash
python -m knowledge_base_builder "your input here"
```
